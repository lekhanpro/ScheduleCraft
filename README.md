# ScheduleCraft

ScheduleCraft is a visual timetable builder and schedule analyzer built with Next.js 14, TypeScript, Tailwind CSS, Zustand, Framer Motion, and Recharts.

## Features

- Weekly and day timetable views
- Real-time conflict detection
- Free-gap analysis
- Study window recommendations
- Workload heatmap
- localStorage persistence
- JSON import and export

## Local Development

```bash
npm install
npm run dev
```
